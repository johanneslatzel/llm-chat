# Hooks

Hooks let you react to events in the chat and tool lifecycle.

- **[Chat hooks](chats/chat-hooks.md)** — observe streaming output (chunk, reasoning, finish) and subscribe to messages by role/regex
- **[Tool hooks](tools/tool-hooks.md)** — before, after, and error hooks on tool execution
- **[Interrupt](services/interrupt.md)** — push events into the conversation flow from outside (e.g. timer expiry)
